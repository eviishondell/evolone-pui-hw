Color Picker Project

Building accessibility-driven tools for WatchOS to enhance user experience for everyone.

Accessible color picker prototype designed for color-blind users on WatchOS.

Project Aim

This project aimed to prototype a color picker tool for WatchOS that was both functional and accessible, focusing on meeting the needs of color-blind users.The design ensured compliance with Apple’s accessibility standards while offering seamless interaction with WatchOS apps.
